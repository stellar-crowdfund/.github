# Stellar Crowdfunding Platform

A decentralized crowdfunding platform built on the Stellar Network that allows transparent and secure donations using blockchain technology.

## Overview

This project demonstrates how blockchain can power a transparent fundraising platform where all donations are recorded on-chain.

## Repositories

| Repository | Description |
|------------|-------------|
| stellar-crowdfunding-backend | Backend API for campaigns, wallets, and donations |
| stellar-crowdfunding-frontend | Frontend application |
| stellar-crowdfunding-docs | System architecture and documentation |

## Features

- Blockchain-based donations
- Transparent funding records
- Wallet creation
- Campaign management
- Real-time transaction tracking

## Technology Stack

Backend
- Node.js
- NestJS
- PostgreSQL

Frontend
- Next.js
- React

Blockchain
- Stellar SDK
- Horizon API
